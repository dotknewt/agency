# Project State
Keep `STATE.md` at the project root current.
Don't repeate CLAUDE.md content, otherwise it must cover:
- What: one-paragraph summary of the project goal
- How: minimal usage instructions (enough to run it)
- Done: completed features/tasks
- WIP: what is actively being worked on
- ToDo: known remaining work
- Decisions: key design choices and why
